# Examen Tema 1: Introducción a la Graficación por Computadora

Este proyecto es una aplicación web que utiliza la API Canvas 2D de HTML5 para renderizar un paisaje generado completamente mediante programación con JavaScript, compuesto por más de 30 figuras geométricas básicas.

## 🚀 Características
* **Dibujo algorítmico:** Uso de primitivas como `fillRect`, `arc`, `lineTo` y `moveTo`.
* **Interfaz amigable:** Estructura construida sobre **HTML5** y estilizada con **Bootstrap 5**.
* **Código organizado:** Modularización en JavaScript, separando cada elemento del dibujo en funciones específicas.
* **Diseño Responsivo:** Visualización adaptable y comparativa (dibujo vs imagen original).

## 🛠️ Tecnologías utilizadas
* HTML5
* CSS3 & Bootstrap 5
* JavaScript (Canvas 2D API)
